Asana Copy Bookmarklet

This is a simple bookmarket to write a link for the currently open Asana task to your clipboard.

It will copy both the task name and URL as an HTML anchor link that you can use to easily paste into Slack, Gmail, etc.

See the code on GitHub.

Step 1

Drag the link/button below into your bookmarks bar!

Copy Asana Link

Step 2

Visit any Asana Page with a task open and click the bookmark. If successful, you'll see the page flash like a camera.

Step 3

Paste the link as needed anywhere that supports basic html rich text. Slack, Gmail, Google Docs, etc.
